# Legacy Experiment Entrypoints

These scripts are archived because they contain earlier experiment settings such as
`q_req=0.60`, `depth_req=0.40`, `timesteps=1000000`, `n_steps=4096`, and
`batch_size=256`. They are kept for provenance only and should not be used for
the current paper results.

- `scripts/run_eiffel_crpo_compare.sh`
- `scripts/run_fushimi_crpo_compare.sh`
